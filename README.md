# Project Latte

Project Latte is a cozy café landing page built with HTML, CSS, and JavaScript.

The page includes responsive styling, interactive music controls, accessible button states, and four café-inspired audio moods.

## Features

- Responsive layout
- Four selectable café soundtracks
- Play, pause, resume, and track switching
- Accessible `aria-pressed` button states
- Reusable CSS button components
- Mobile-friendly design
- Deployed with GitHub Pages

## Built With

- HTML5
- CSS3
- JavaScript
- GitHub Pages

While building Project Latte, I practiced:
